## 📖 وصف المستودع

هذا المستودع يوثق رحلة كاملة لتعلم لغة JavaScript من الأساسيات إلى المفاهيم المتقدمة،
يشمل الكائنات، التعامل مع DOM، البرمجة الكائنية، البرمجة غير المتزامنة (Promises، Fetch)،
وحل التحديات البرمجية الإبداعية.

## 🚀 ماذا ستجد هنا؟

### المفاهيم الأساسية
- **المتغيرات وأنواع البيانات** - التعامل مع الأعداد والنصوص والقيم المنطقية
- **الدوال** - تعريف الدوال، التعبيرات الدالية، دوال السهم
- **الكائنات** - إنشاء الكائنات، الخصائص، والطرق
- **المصفوفات** - جميع طرق المصفوفات مع أمثلة عملية

### أمثلة عملية
- **نظام حساب بنكي** - إيداع وسحب مع تحقق من الرصيد
- **كتالوج منتج** - كائنات متداخلة مع دوال لحساب الأسعار
- **التحقق من النماذج** - تحقق فوري مع أنماط Regex
- **حقول OTP** - انتقال تلقائي بين الحقول
