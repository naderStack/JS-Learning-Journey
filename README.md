# JS-Learning-Journey

A comprehensive JavaScript exploration repository covering fundamentals, OOP, DOM manipulation, asynchronous programming (Promises, Fetch), and creative coding challenges from basic syntax to advanced concepts.

## 📚 Repository Description 
This repository documents a complete JavaScript learning journey from fundamentals to advanced concepts, including objects, DOM manipulation, OOP, Promises, error handling, recursion, and creative problem-solving through hands-on coding examples.

## 🚀 What's Inside

### Core JavaScript Concepts
- **Variables & Data Types** - Working with numbers, strings, booleans, and type conversion
- **Operators** - Arithmetic, comparison, logical operators with creative challenges
- **Functions** - Declarations, expressions, arrow functions, and callbacks
- **Objects** - Creating objects, methods, nested objects, and object manipulation
- **Arrays** - Methods like slice, splice, concat, sort, reverse, and array destructuring

### Advanced Topics
- **Closures** - Understanding scope and closure behavior
- **`this` Keyword** - Differences between regular functions and arrow functions
- **OOP (Object-Oriented Programming)** - Constructor functions and ES6 classes
- **DOM Manipulation** - Creating dynamic forms, OTP inputs, and interactive elements
- **Asynchronous JavaScript** - Callbacks, Promises, and Fetch API
- **Error Handling** - Try-catch blocks for robust code
- **Recursion** - Understanding recursive functions

### Practical Examples
- **Bank Account System** - Deposit and withdrawal with validation
- **Product Catalog** - Nested objects with methods for pricing and inventory
- **Form Validation** - Real-time input validation with regex patterns
- **OTP Input Fields** - Auto-focus navigation between inputs
- **User Settings** - Object merging and property management
- **Data Fetching** - Working with JSON and external APIs
